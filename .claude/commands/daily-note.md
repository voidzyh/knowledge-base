请帮我创建今日学习笔记。

## 执行步骤

### 1. 检查今日笔记
首先检查今日笔记是否已存在：
- 文件路径格式：`daily-notes/YYYY-MM-DD.md`
- 如果已存在，打开现有笔记
- 如果不存在，使用模板创建新笔记

### 2. 使用模板创建
如果需要创建新笔记：
- 使用 `templates/daily-note-template.md` 模板
- 自动填充以下变量：
  - `{{DATE}}`: 今日日期 (YYYY-MM-DD)
  - `{{DAY_OF_WEEK}}`: 星期几
  - `{{TAGS}}`: 推荐标签

### 3. 提供智能建议
分析以下信息并添加到笔记中：

#### 3.1 分析昨日学习
- 读取昨日的每日笔记文件
- 识别未完成的学习目标
- 提取需要延续的学习主题

#### 3.2 识别今日复习任务
扫描知识库，识别今日需要复习的内容：
- 查找 `review_dates` 字段包含今日的所有知识条目
- 按掌握程度排序（低掌握度的优先）
- 显示复习提醒列表

#### 3.3 推荐今日学习主题
基于以下因素推荐：
- 昨日学习的延续
- 长期学习目标
- 当前知识盲区
- 用户兴趣和趋势

### 4. 预填充今日目标
在笔记中添加：
- **延续学习**: 从昨日延续的主题
- **复习任务**: 今日需要复习的主题列表
- **新学习**: 推荐的新主题
- **实践项目**: 建议的实践或项目

### 5. 显示学习统计
显示本周/本月的学习统计：
- 已完成的学习目标数
- 新增知识条目数
- 完成的复习次数
- 学习效率评估

## 注意事项

1. **日期计算**:
   - 使用系统当前日期
   - 自动计算昨日的日期
   - 正确处理月份和年份边界

2. **错误处理**:
   - 如果模板文件不存在，使用默认结构
   - 如果昨日笔记不存在，跳过延续学习建议
   - 如果没有到期复习，显示"无紧急复习任务"

3. **交互友好**:
   - 提供清晰的选项 (y/n)
   - 自动打开编辑器（如果用户选择 y）
   - 保存笔记文件路径供后续使用

4. **集成工具**:
   - 可以调用现有的 `tools/create-daily-note.sh` 脚本
   - 利用 Git 获取历史记录
   - 使用 ripgrep 搜索相关内容
