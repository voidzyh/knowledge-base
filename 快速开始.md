# 🚀 快速开始指南

欢迎使用你的个人知识库！这个指南将帮助你快速上手。

## 1️⃣ 第一次使用

### 创建今天的笔记
```bash
# 在 daily-notes 目录下创建今天的学习笔记
cd /Users/zyhz/KnowledgeBase/daily-notes
touch 2025-12-10.md
```

然后使用 `templates/daily-note-template.md` 作为模板开始记录。

### 建立你的第一个知识条目
在相应的主题目录下创建新的知识条目：
- AI 相关知识 → `ai-ml/llm/`
- 编程知识 → `programming/swift/` 或 `programming/python/`
- 技术知识 → `tech-knowledge/frontend/`

## 2️⃣ 日常使用流程

### 每天学习时
1. 打开 `daily-notes/{{今天日期}}.md`
2. 记录学到的新知识
3. 标注重要概念和待办事项

### 每周整理时
1. 回顾本周的每日笔记
2. 将重要知识点整理成独立的知识条目
3. 更新复习计划

### 每月复习时
1. 打开复习计划模板
2. 按照间隔重复法复习
3. 更新掌握程度评估

## 3️⃣ 实用技巧

### 标签系统
使用统一的标签格式：
- `#AI` `#LLM` `#Agent` - AI相关
- `#Swift` `#Python` `#JavaScript` - 编程语言
- `#iOS` `#WebDev` `#DevOps` - 技术领域
- `#每日学习` `#重要` `#待深入` - 学习状态

### 文件命名规范
- 每日笔记：`YYYY-MM-DD.md`
- 知识条目：使用英文和连字符，如 `transformer-architecture.md`
- 复习计划：`review-YYYY-MM.md`

### 交叉引用
在知识条目之间建立链接：
```markdown
- 参见：[相关概念](相关文件名.md)
- 前置知识：[基础概念](基础文件名.md)
- 深入阅读：[高级主题](高级文件名.md)
```

## 4️⃣ 推荐的复习节奏

### 艾宾浩斯遗忘曲线
- **20分钟后** - 遗忘42%
- **1小时后** - 遗忘56%
- **1天后** - 遗忘74%
- **1周后** - 遗忘77%
- **1个月后** - 遗忘79%

### 复习时间安排
- **学习后1天** - 第1次复习
- **学习后3天** - 第2次复习
- **学习后1周** - 第3次复习
- **学习后2周** - 第4次复习
- **学习后1个月** - 第5次复习

## 5️⃣ 高级功能

### 思维导图
对于复杂概念，可以：
1. 创建简单的 ASCII 艺术思维导图
2. 使用在线工具制作，保存图片到 `resources/diagrams/`
3. 在知识条目中引用

### 代码片段管理
- 将常用代码保存到 `resources/code-snippets/`
- 按语言和功能分类
- 在知识条目中引用

### 资源库建设
- `resources/articles/` - 保存重要的文章
- `resources/videos/` - 记录有用的视频
- `resources/tools/` - 整理工具清单
- `resources/books/` - 书籍笔记

## 7️⃣ 备份和同步

建议定期备份知识库：
```bash
# 压缩整个知识库
tar -czf knowledge-backup-$(date +%Y%m%d).tar.gz /Users/zyhz/KnowledgeBase

# 或使用 git 管理（如果需要版本控制）
cd /Users/zyhz/KnowledgeBase
git init
git add .
git commit -m "初始知识库"
```

## 🎯 成功秘诀

1. **坚持每日记录** - 不要中断，哪怕只是简单几行
2. **定期整理** - 每周花时间整理和分类
3. **主动回忆** - 合上笔记，尝试回忆关键概念
4. **实践应用** - 在实际项目中使用学到的知识
5. **分享输出** - 写博客或教给他人，巩固理解
