# 个人知识库

这是一个系统化的知识存储和管理系统，帮助你收集、整理和复习每天学到的知识。

## 📁 目录结构

### 按主题分类
- **`ai-ml/`** - AI 和机器学习相关知识
  - `llm/` - 大语言模型
  - `agent/` - AI 智能体
  - `mcp/` - 模型上下文协议
  - `nlp/` - 自然语言处理
  - `computer-vision/` - 计算机视觉

- **`programming/`** - 编程技术
  - `swift/` - Swift 和 iOS 开发
  - `python/` - Python 编程
  - `javascript/` - JavaScript 和前端
  - `algorithms/` - 算法和数据结构
  - `frameworks/` - 框架和工具

- **`tech-knowledge/`** - 技术知识
  - `frontend/` - 前端技术
  - `backend/` - 后端技术
  - `devops/` - DevOps 和运维
  - `database/` - 数据库
  - `architecture/` - 系统架构

- **`tools/`** - 工具和软件
- **`books/`** - 书籍笔记和总结
- **`resources/`** - 有用的资源链接

### 按时间归档
- **`daily-notes/`** - 每日学习笔记
  - 按日期命名：`2025-12-10.md`

## 📝 使用方法

### 每日学习笔记
1. 在 `daily-notes/` 目录下创建以日期命名的文件
2. 使用每日笔记模板记录当天学到的重要知识
3. 定期回顾和整理

### 知识条目管理
1. 将相关知识按主题分类存储到对应目录
2. 使用知识条目模板标准化格式
3. 建立交叉引用和标签系统

### 复习策略
- **每日回顾**：花 10-15 分钟回顾当天笔记
- **每周整理**：将零散笔记整理成系统性知识
- **每月复习**：全面回顾本月学习内容
- **季度总结**：形成季度学习报告

## 🔄 知识循环

```
学习 → 记录 → 整理 → 复习 → 应用 → 深化
```

## 📋 模板文件

- `templates/daily-note-template.md` - 每日笔记模板
- `templates/knowledge-entry-template.md` - 知识条目模板
- `templates/review-plan-template.md` - 复习计划模板
