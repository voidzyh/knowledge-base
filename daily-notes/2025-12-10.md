## 📚 学习内容

### 1. AI Agent 基础概念
**关键概念**：
- AI Agent：能够感知环境、做出决策并采取行动以实现特定目标的自主系统
- 自主性：不需要持续的人为指令，能够独立工作
- 感知-决策-执行循环：Agent 的核心工作模式

**重要发现**：
- AI Agent 代表了 AI 从"能说会道"到"能谋善断"的质的飞跃
- Agent 通常由大语言模型驱动，结合规划、记忆、感知和行动模块
- 2025年被称为 AI Agent 元年，技术进入主流市场

### 2. LLM、MCP、Agent 关系解析
**关键概念**：
- LLM（大型语言模型）：基础的"大脑"，擅长语言理解和推理
- MCP（模型上下文协议）：标准化的"接口"，解决 AI 与外部世界交互问题
- Agent：整合 LLM 和 MCP 的完整"执行者"

**协作流程**：
1. LLM 负责理解和推理
2. MCP 提供标准化的工具访问接口
3. Agent 实现感知-决策-执行的完整闭环

## 💡 灵感与思考
- Agent 的核心价值在于将"思考"转化为"行动"
- MCP 解决了 AI"孤立"的问题，是连接现实世界的重要桥梁
- 未来可能出现更多基于 Agent 的应用场景
